static void WebScrapeWidget.Utilities.FileSystemUtilities.ValidateDirectory ( string path,
bool shallExist = true )
static

Checks if specified file system entry exists and is a directory. If either of above cases would not be fulfilled, according exception will be thrown.

Parameters
pathPath to file system entry, which shall be validated.
shallExistSpecifies if provided file system entry shall exists within file system as directory. Useful, when there is a need to check if specified path is not already in use by other directory in file system.
Exceptions
ArgumentExceptionThrown, when at least one argument will be considered as invalid.
DirectoryNotFoundExceptionThrown, when provided file system entry does not exist in file system as directory.
IOExceptionThrown, when provided file system entry exists within file system as directory but shall not.

IEnumerable< string > validExtensions )
static

Checks if extension of specified file system entry is equal to expected one. If validation condition would not be fulfilled, according exception will be thrown.

Check is based entirely on provided path - method does not check if specified entry exists within file system.

Parameters
pathPath to file system entry, which shall be validated.
validExtensionsCollection of extensions, which shall be considered as valid for validated file.
Exceptions
ArgumentExceptionThrown, when at least one argument will be considered as invalid.
IOExceptionThrown, when specified file system entry has invalid extension.
