Detailed Description

Repository, that provided access to a pool of data sources.

Keep in mind, that initially data from all sources contained by the repository is not gathered and attempt to access it will result in raised exception.

◆ StartPeriodicDataGathering()

async Task WebScrapeWidget.DataGathering.Repositories.DataSourcesRepository.StartPeriodicDataGathering ( )

Starts the process of infinite, periodic data gathering from all sources cantoned by repository.

Refresh rate of every data source is individual. Maximal possible refresh rate for a data source is 1 second - refresh rates not divisible by 1 second will be rounded.

Returns
Task corresponding to status periodic data gathering process, which never be completed as process is infinite.
