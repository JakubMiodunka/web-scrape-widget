Detailed Description

Utility, which swaps the culture used by the thread, executes a block of code and reverts the culture change. Designed to use with 'using' statement.

CultureInfo.CurrentCulture = new CultureInfo("en-GB")
// Here 'en-GB' culture is being used.
using(var cultureContext = new CultureContext("pl-PL"))
{
// Here 'pl-PL' culture is being used.
}
// Here 'en-GB' culture is being used.
